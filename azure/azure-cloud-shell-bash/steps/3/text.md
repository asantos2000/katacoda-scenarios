Create a new resource group in WestUS named "MyRG".
`az group create --location westus --name MyRG`{{copy}}


You will get following output:

```
{
  "id": "/subscriptions/<id>/resourceGroups/MyRG",
  "location": "westus",
  "managedBy": null,
  "name": "MyRG",
  "properties": {
    "provisioningState": "Succeeded"
  },
  "tags": null,
  "type": null
}
```
